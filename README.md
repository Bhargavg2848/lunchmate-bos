# Lunchmate BOS

Lunchmate BOS is a mobile-first, low-cost POS and order management system for small food businesses. It is designed to start lean (5–10 daily clients) and scale to 50+ clients quickly using free-tier infrastructure.

## Phase 1 Scope (Current)
- Authentication and Role-Based Access
- Customer Management
- Catalog Management
- Orders and Billing
- Basic dashboard and daily operations

## Tech Stack
- Next.js + TypeScript + Tailwind CSS
- Supabase (Postgres, Auth, Realtime, Storage)
- Vercel deployment
- npm package manager

## Repository Structure (planned)
- `docs/` – planning and architecture documents
- `web/` – Next.js app (to be scaffolded next)
- `supabase/` – migrations and SQL policies (to be added)

## Getting Started (will be enabled after scaffold)
```bash
# clone
git clone https://github.com/Bhargavg2848/lunchmate-bos.git
cd lunchmate-bos

# install
npm install

# run
npm run dev
```
