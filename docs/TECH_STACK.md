# Tech Stack (Budget-Optimized)

## Core
- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS
- **Backend/Data:** Supabase (Postgres + Auth + Realtime + Storage)
- **Deployment:** Vercel (frontend), Supabase hosted backend
- **Observability:** Sentry (free tier, optional in first week)

## Why this stack
- Fast development speed
- Strong free tiers
- Good scalability for early-stage growth
- Minimal infra maintenance overhead

## Package Manager
- npm

## Future Add-ons
- Playwright for critical end-to-end tests
- Queue/background jobs if order volume spikes
- AI assistant features from phase 4 onward
